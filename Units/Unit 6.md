### Grammar

| Subject   | Verb **(Devoir)** </br> (must) |
| --------- | ------------------------------ |
| Je        | **dois**                       |
| Tu        | dois                           |
| Il/elle   | doit                           |
| Nous      | d**evons**                     |
| Vous      | d**evez**                      |
| Ils/elles | **doivent**                    |
